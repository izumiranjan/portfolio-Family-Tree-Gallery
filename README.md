# Family Tree & Gallery

A simple web-based Family Tree and Gallery project built using HTML, CSS, and JavaScript. This application helps visualize family relationships and store family memories in a structured and interactive way.

## 🚀 Features
- Interactive family tree structure  
- Display family member details  
- Photo gallery for family memories  
- Simple and clean user interface  
- Easy navigation between members  
- Responsive web design  
- Built with pure HTML, CSS, and JavaScript  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

## 📁 Project Structure

Family-Tree-Gallery/
│── index.html
│── family-tree.html
│── gallery.html
│── style.css
│── script.js
│── assets


## ▶️ How to Run
1. Download or clone this repository
2. Open the project folder
3. Double click `index.html`
4. Project will open in your browser

## 🎯 Purpose
This project is designed to organize family information, visualize relationships, and store memorable family photos in one place. It can be used for learning, college projects, or personal family documentation.
